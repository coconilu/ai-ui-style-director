# Context Questions

Ask only what is required to choose a useful UI direction.

For a vague new-site request, ask:

1. Is this a new website or a redesign?
2. What type of surface is it: landing page, SaaS product site, dashboard, docs site, portfolio, ecommerce, or internal tool?
3. Who is the primary audience?
4. What is the primary goal: signup, purchase, booking, product explanation, show work, or daily operation?
5. Should the result feel sparse, balanced, or information-dense?

For redesigns, also ask for one source:

- URL
- screenshot
- local project path
- existing brand/style assets

If the user already supplied enough context, do not ask these questions. Recommend styles immediately.
